    Python Detectors

    Using PyCall, we can easily integrate existing python outlier detection algorithms. Currently, almost every PyOD algorithm is integrated and can thus be easily used directly from Julia. As a naming convention, every python detector is named Py$NAME.

    PyABOD

    # OutlierDetection.PyABOD — Type.

    PyABOD(n_neighbors = 5,
          method = "fast")

    https://pyod.readthedocs.io/en/latest/pyod.models.html#module-pyod.models.abod

    source
